Saffron API Documentation
=========================


Base URL: http://{saffron_ip_address}:8080/api/v1

API Routes in place:

* GET /run

`This API call returns all previous saffron runs`

* POST /run

`This API call returns all previous saffron runs`

`This API call returns all previous saffron runs`
