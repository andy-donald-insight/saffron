OATS ("Open Source Assistive Technology Software") is a source code repository or "forge" for assistive technology software. It was launched in 2006 <ref>Steve Lee: "OATSoft, a new service and a landmark for Accessibility", 6 May 2006.</ref> with the goal to provide a one-stop “shop” for end users, clinicians and open-source developers to promote and develop open source assistive technology software. It also allows users to find other software that is free but not open source,<ref name="projectSummary">OATS: Project Summary.</ref> and open-source software that is developed elsewhere, for example NVDA,<ref>OATS: NVDA (Non Visual Desktop Access).</ref> Orca<ref>OATS: Orca.</ref> and FireVox.<ref>OATS: Fire Vox.</ref>

The OATS website was launched at the end of the OATS project, a one-year pilot project that ended in March 2006, and claimed to be the first open-source repository dedicated to assistive technology.<ref name="projectSummary" /><ref>OATS: OATS Launch Press Release, 3 May 2006.</ref> In April 2006, the British Computer Society (BCS) announced that it was backing the OATS project;<ref>Justin Richards: "BCS backs open source project to boost software for disabled users" Computer Weekly, Thursday 20 April 2006.</ref> in August 2006, the British Computer Society's Open Source Specialist Group organised a meeting about the project.<ref>"OATS Meeting Aftermath", BCS Open Source Specialist Group Blog, Friday, August 11th 2006.</ref>

The OATS project was made up of five partner organisations co-ordinated by the ACE Centre.<ref>OATS: OATS Consortium.</ref> The OATS repository wants to offer an efficient and intuitive way to access good quality assistive technology.<ref>OATS: Project Context.</ref> The OATS repository offers the following facilities to developers:<ref>OATS: Hosting on OATS.</ref>

  • project pages to document and manage a software project,
  • a code repository (Subversion),
  • a project management system (Trac or Poi).
